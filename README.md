# flutter_loginSignup-phpMyAdmin

Flutter login and Signup example using phpMyAdmin.


## Source Code Includes:

  * [x] API(written in php)
  * [x] Database File
  * [x] Screenshots
  * [x] Fully working login and Signup code in dart
  * [x] Toast
  * [x] Materiall design of TextField
